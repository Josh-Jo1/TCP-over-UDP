# TCP-over-UDP



## Usage

Terminal 1
```
python receiver.py
```

Terminal 2
```
python sender.py
```

## Development Process

1. Create basic sender and receiver, connected over UDP.
2. Create a Packet class to send header + data as bytes through sockets.


## Versions

Python 3.10.2

Ubuntu 22.04.3 LTS

WSL2 5.15.133.1
